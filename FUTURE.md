## ✒ 未来版本的新特性 (Features in future versions)

### v0.2.x

* [ ] 支持 MD5 散列算法
* [ ] 支持 SHA1 散列算法
* [ ] 支持 SHA256 等散列算法
* [ ] 支持 SHA512 等散列算法
* [ ] 支持 HMAC 等散列算法

### v0.1.x

* [ ] 支持 RSA 非对称加密算法

### v0.0.x

* [ ] 支持 DES/3DES/AES 对称加密算法
* [ ] 支持 ECB/CBC/OFB/CFB/CTR 等分组模式
* [x] 支持 PKCS5/PKCS7/ZERO 等填充方式
* [x] 支持 HEX/BASE64 等编解码算法
